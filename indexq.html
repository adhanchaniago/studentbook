Deskripsi Studentbook

Studentbook.com

Portal pendidikan www.studentbook.com adalah sebuah web yang digunakan di bidang pendidikan yang melibatkan sekolah, guru, siswa, dan orang tua siswa.


Semua unsur di lingkungan pendidikan tersebut dapat saling berinteraksi satu dengan lainnya. Web ini bukan dibuat untuk satu sekolah tertentu melainkan dapat digunakan untuk seluruh sekolah yang telah tergabung/terdaftar sebagai web ini.

A. Fitur Sistem Akademik Registrasi Sekolah, pertama kali sebelum suatu sekolah dapat menggunakan web ini, maka sekolah tersebut harus melakukan registrasi terlebih dahulu melalui website ini. Setelah registrasi maka admin web www.studentbook.com akan mengaktifkan akun sekolah tersebut setelah seluruh administrasi diselesaikan. Setiap sekolah akan mendapatkan userID dan password untuk admin sekolah.

Sekolah anda belum terdaftar?

    Untuk dapat login ,maka Anda harus terdaftar terlebih dahulu di sekolah anda. Pihak Admin Sekolah lah yang akan mendaftarkan akun anda. Jika Sekolah anda sudah terdaftar ,silahkan hubungi admin sekolah untuk mendaftarkan user ID dan pasword.
